# Donate for Dreams

Donate for Dreams is a static web project focused on simulating a donation platform
for social causes, emphasizing front-end structure, visual organization, and user experience.

The project explores how a donation-oriented website can present campaigns,
informational sections, and user navigation in a clear and accessible way.
